# Town Runner – Prototype 3

This project is Prototype 3 from my Unity learning journey.
It is based on Unity’s Create with Code course.

## Features

- Endless side‑scrolling runner gameplay​
- Player jump mechanic (Space)
- Dynamic repeating background system​
- Particle effects for dirt splatter and explosions
- Player animations for running, jumping and collisions​​
- Background music and sound effects for jumps and crashes​​
- SpawnManager for obstacles
- Simple game over logic with restart functionality ​(Escape)

## Purpose
This prototype focuses on:
- player animation
- sound effects and background music
- particle systems
- basic obstacle spawning
- simple endless runner mechanics
